# hi2gage Homebrew Tap

Custom Homebrew formulas and casks.

## Installation

```bash
brew tap hi2gage/hi2gage
```

## Available Casks

| Cask | Description |
|------|-------------|
| `buildsignal` | Analyze Xcode build logs and warnings |

### Install BuildSignal

```bash
brew install --cask buildsignal
```
